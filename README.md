# SecureInspect

A collection of lightweight cybersecurity tools built in Python for security analysis, monitoring, and threat awareness.

## Available Tool

### QR Code Safety Scanner

A Python-based tool that analyzes QR code content and detects potentially suspicious indicators such as:

- Non-HTTPS URLs
- URL shorteners
- IP-based URLs
- Suspicious top-level domains (TLDs)
- Phishing-related keywords

Each scanned QR code is evaluated and assigned a risk score along with a safety status.
